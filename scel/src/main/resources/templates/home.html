Spring Boot facilita a criação autonoma de aplicações que podem rapidamente ser colocada em produção. O Spring Boot torna disponivel um conjunto de bibliotecas de terceiros que podem ser utilizados com pouca configuração.
The Spring Tool Suite é um ambiente de desenvolvimento baseado em Eclipse que está customizado para implementação de aplicações Spring. O STS prove uma estrutura de programação integrada para debugar, executar e entregar uma aplicação Spring incluindo a configuração com um servidor de aplicações, Git,
Thymeleaf é um framework de programação da interface de comunicação para aplicações Java web. Atualmente existem varios frameworks de integração da GUI com o servidor web.
WebJars são bibliotecas utilizadas no lado do cliente empactodas dentro de arquivos Jar (e.g. jQuery ; Bootstrap).
Hibernate ORM facilita o desenvolvimento de aplicações para manipulação de dados. O framework Hibernate é uma ferramenta de mapeamento Object/Relational Mapping (ORM) que permite a persistencia utilizando banco de dados relacional (via JDBC).
Hibernate implementa a especificação JPA.
Bootstrap fornece uma estrutura para o desenvolvimento de HTML, CSS e JS. Permite a prototipagem rapida de para validar o design da interface de sistemas de softwareo.
